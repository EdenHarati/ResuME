**Job Title: Java Developer**\
\
**Location:** San Francisco, CA\
\
**Company:** Tech Innovators Inc.\
\
**Job Type:** Full-Time\
\
**Salary:** Competitive, based on experience\
\
**Job Overview:**\
\
Tech Innovators Inc. is seeking a talented and motivated Java Developer to join our dynamic software development team. The successful candidate will be responsible for designing, developing, and maintaining high-performance Java applications. You will work closely with cross-functional teams to deliver innovative software solutions that meet our clients' needs.\
\
**Key Responsibilities:**\
\
- **Design and Development:** Develop, test, and deploy high-quality Java applications using Java SE/EE, Spring Framework, and Hibernate.\
- **API Integration:** Create and maintain RESTful APIs and integrate them with third-party services.\
- **Performance Optimization:** Analyze and improve application performance, ensuring efficient database queries and optimal code execution.\
- **Security Implementation:** Implement security best practices, including authentication and authorization using Spring Security.\
- **Code Review and Mentorship:** Conduct code reviews to maintain code quality and provide mentorship to junior developers.\
- **Agile Collaboration:** Participate in Agile/Scrum meetings, including sprint planning, daily stand-ups, and retrospectives.\
- **Documentation:** Write and maintain technical documentation for software design, development processes, and deployment procedures.\
- **Troubleshooting:** Debug and resolve technical issues in a timely manner to ensure smooth application operation.\
\
**Required Qualifications:**\
\
- **Education:** Bachelor's degree in Computer Science, Information Technology, or a related field.\
- **Experience:** At least 3-5 years of professional experience in Java development.\
- **Technical Skills:**\
  - Proficiency in Java SE/EE\
  - Experience with Spring Framework (Spring Boot, Spring MVC, Spring Security)\
  - Strong knowledge of Hibernate ORM\
  - Experience with RESTful web services and API integration\
  - Familiarity with front-end technologies (HTML, CSS, JavaScript)\
  - Knowledge of databases (MySQL, PostgreSQL, Oracle)\
  - Experience with version control systems (Git, SVN)\
  - Familiarity with build tools (Maven, Gradle)\
  - Understanding of CI/CD processes (Jenkins, Docker)\
  - Exposure to cloud platforms (AWS, Azure) is a plus\
- **Soft Skills:**\
  - Excellent problem-solving skills\
  - Strong communication and teamwork abilities\
  - Ability to work independently and manage multiple tasks\
  - Attention to detail and commitment to quality\
\
**Preferred Qualifications:**\
\
- **Certifications:** Oracle Certified Professional, Java SE 11 Developer or similar certification.\
- **Experience:** Previous experience in a senior developer role or leading development teams.\
- **Knowledge:** Understanding of microservices architecture and containerization (Docker, Kubernetes).\
\
**Benefits:**\
\
- Competitive salary and performance-based bonuses\
- Comprehensive health, dental, and vision insurance\
- 401(k) with company match\
- Generous paid time off and holidays\
- Professional development opportunities\
- Flexible working hours and remote work options\
- Collaborative and inclusive work environment\
\
**How to Apply:**\
\
Interested candidates are encouraged to send their resume and cover letter to careers@techinnovators.com with the subject line "Java Developer Application \'96 [Your Name]". Please include links to your LinkedIn profile and GitHub repository (if applicable).\
\
**About Tech Innovators Inc.:**\
\
Tech Innovators Inc. is a leading software development company specializing in cutting-edge technology solutions for businesses worldwide. Our mission is to empower clients with innovative software that drives success and growth. Join our team of passionate professionals dedicated to making a difference through technology.\
